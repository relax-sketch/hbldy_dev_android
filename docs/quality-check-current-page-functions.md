# 草原监测质检当前页面功能说明

日期：2026-05-27

本文描述当前 App 中质检相关页面的功能和交互，用于后续前端视觉重设计时理解现有页面。本文只记录当前行为。

## 1. 页面入口与导航

质检流程从“草原监测质检”页面进入，主要页面顺序为：

1. 数据目录选择页
2. 选择质检范围页
3. 质检进行中页
4. 质检结果页
5. 样地详情页

顶部标题会随当前页面变化：

| 页面 | 标题 |
| --- | --- |
| 数据目录选择页 | 草原监测质检 |
| 选择质检范围页 | 选择质检范围 |
| 质检进行中页 | 质检进行中 |
| 质检结果页 | 质检结果 |
| 样地详情页 | 样地详情 |

除首页和进行中页外，顶部左侧提供“返回”。Android 系统返回键遵循同一层级：范围页返回首页，结果页返回范围页，详情页按来源返回结果页或范围页。

## 2. 数据目录选择页

数据目录选择页用于选择或重新扫描“草原监测/数据”目录。应用通过系统目录选择器获取目录权限，并递归扫描其中的 `.zdb` 文件。

页面功能：

1. “选择数据目录”：打开系统目录选择器。
2. “重新扫描当前目录”：在已保存目录存在时重新扫描。
3. 扫描中显示进度提示“正在查找并读取 ZDB 结构...”。
4. 扫描完成后显示可读取 ZDB 数量、索引样地数量、无法使用文件数量。
5. 当存在可质检样地时，显示“选择质检范围”按钮进入范围页。

扫描和质检查询均以只读方式读取 ZDB。

## 3. 选择质检范围页

范围页用于筛选和选择样地，并启动质检。

页面右上角有两个小开关：

| 开关 | 当前行为 |
| --- | --- |
| 全量 | 打开后清空并禁用区县下拉、样地编号搜索和样地列表选择，开始质检时检查全部已索引样地 |
| 测试 | 控制结果和详情中是否展示已通过规则 |

范围页筛选控件：

1. 区县下拉框：默认包含“全域”，其余选项来自已索引样地的区县标签。
2. 样地编号搜索：输入编号后筛选样地列表。
3. 样地列表：展示匹配样地，点击后选中该样地。
4. 匹配数量：显示当前筛选条件下的样地数量。

普通模式下必须点选一个样地后才能点击“开始质检”。普通模式启动后检查单个样地，完成后直接进入该样地详情页。

全量模式下不需要点选样地。“开始质检”会检查全部已索引样地，完成后进入质检结果页。

## 4. 质检进行中页

进行中页展示当前质检进度。

页面内容：

1. 线性进度条。
2. 已完成样地数和总样地数。
3. 当前正在质检的样地编号。
4. 只读查询说明。
5. “取消本次质检”按钮。

取消后会保留已完成样地的结果，并进入可查看结果的后续页面。

## 5. 质检结果页

结果页用于查看批量质检后的总体统计和样地列表。

顶部统计包括：

| 统计项 | 含义 |
| --- | --- |
| 强制性 | 未忽略的强制性问题总数 |
| 提示性 | 未忽略的提示性问题总数 |
| 已忽略 | 已标注忽略的问题总数 |
| 已跳过规则 | 因表或字段不兼容等原因未执行的规则数 |
| 已通过规则 | 测试模式打开时显示，通过的规则数 |
| 已执行规则 | 测试模式打开时显示，实际执行的规则数 |

样地结果列表按样地展示问题摘要。每个样地卡片显示样地编号、强制性问题数、提示性问题数、已忽略数量，并在存在跳过规则时显示跳过数量。测试模式打开时还显示通过规则数量。

点击某个样地卡片进入该样地详情页。

页面底部操作：

1. “再次质检”：按上一次范围重新执行质检。
2. “更改范围”：返回范围页重新选择。

## 6. 样地详情页

样地详情页展示单个样地的质检结果。

标题区域显示：

1. 样地编号。
2. 小字汇总：强制性数量、提示性数量、跳过数量、忽略数量。
3. 区县标签和已执行规则数量。

详情页按状态展示结果：

| 区域 | 内容 |
| --- | --- |
| 待处理强制性问题 | 未忽略的强制性问题 |
| 待处理提示性问题 | 未忽略的提示性问题 |
| 无法执行的规则 | 因表或字段不兼容等原因跳过的规则 |
| 已通过规则（测试模式） | 测试模式打开时展示已通过规则 |
| 已忽略（视为通过） | 用户已标注忽略的问题 |

问题卡片显示规则等级、规则标题、规则编号、表名、说明、定位值和实际值。待处理问题提供“标注忽略”按钮；已忽略问题提供“取消忽略”按钮。

详情页底部操作：

1. “再次质检”：按上一次范围重新执行质检。
2. “返回范围”或“返回汇总”：单样地质检返回范围页，批量质检进入详情后返回汇总页。

## 7. 错误提示

页面顶部会显示当前错误消息，例如无法读取目录、扫描失败、没有可质检样地、未选择样地时尝试开始质检、质检未完成等。

错误提示不会阻止用户重新选择目录、重新扫描或重新选择范围。
